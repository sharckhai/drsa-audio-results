Audio Collections on Sythetic Data

The first two links namely 'Sythetic Class 1', and 'Sythetic Class 2', each contain an examplanary data sample with the four corresponding class specific audio objects. Final samples are combinations of up to four of these distinctive audio object, overlayed with noise and random sounds, to improve generalization of the neural network trained to classify those. Link displaying 'Explanations ...', contain original samples, extracted standard explanations with Layer-wise Relevance Propagation, and the decomposition of those achieved with Disentangled Relevant Subspace Analysis.
