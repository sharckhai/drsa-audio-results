Toy Audio Collections

The first two links namely 'Toy Class 1' and 'Toy Class 2' each contain an examplanary toy sample with the four class specific audio objects which define each class. Final samples are combinations of up to four of these distinctive audio object, overlayed with noise and random sounds to increase generalization of the neural network trained to classify those. The third link 'Explanations', containes extracted standard explanations and the decomposition of those achieved with Disentangled Relevant Subspace Analysis.
